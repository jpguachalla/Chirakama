# Chirakama - Sistema de Gestión para Importadora de Vehículos

## Descripción

Chirakama es un sistema desarrollado en Laravel 11 para la gestión de vehículos de una importadora.

El proyecto implementa una API REST profesional utilizando:

- Laravel Sanctum
- Eloquent ORM
- Middleware personalizado
- Service Pattern
- API Resources
- Form Requests
- Seeders y Factories
- Feature Tests

---

# Tecnologías Utilizadas

- PHP 8.2+
- Laravel 11
- MySQL
- Composer
- Postman
- Laravel Sanctum
